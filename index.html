R-Home is a modular system with two main subsystem. At the core there is the home automation which includes data collection (temperature, humidity, status, image,...), programmed actions (keep temperatue, water plants, lights on/off), instant actions (play music, change ilumination, prepare bath). On the surface the user interface enables full control and setup of all the components at the core. In addition, it can be also use remotly from where ever you are provided you are have internet access.
